Mage Helper

         Mage Helper is a character editor and spellcasting aide for Chronicles of Darkness' Mage the Awakening 2nd Edition. You can make up to 4 characters, save their sheets, and pull them up as you play for quick reference of common rules and mechanics.

         The edit button on the top banner will let you modify any of your character's dots. Any text you can hover over and the cursor presents a question mark mean a side panel will open for more options. Usually this is to edit stats, or reference the book.

         I'm still fairly new to this system, as I'm currently running my first campaign. So some critical features for play may still be missing. This game is pretty complicated so I haven't gone through and added logic for everything. You can make really invalid characters with this app, there are no logic balances. The app currently is just a fancy paper character sheet.
